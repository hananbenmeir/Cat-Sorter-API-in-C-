Cat Sorter API in C# (.NET 6)

This assignment is designed to assess your problem-solving, coding abilities, and understanding of C# functionalities. You'll be building a simple Cat Sorter API that utilizes a unique cat breed sorting algorithm.

Objective:

Develop a C# (.NET 6) console application that exposes a basic API to sort a list of cats based on a specific criterion.

Cat Data Model:

Create a Cat class with the following properties:

  • Name (string): Name of the cat breed
  • Temperament (string): Temperament of the breed (e.g., Playful, Independent)
  • HairLength (string): Hair length of the breed (e.g., Short, Long)

Sorting Algorithm:

Implement a custom sorting algorithm named SortCatsByCompatibility that takes a list of Cat objects and returns a new list sorted based on compatibility. Compatibility is determined by the following rules:

  1. Temperament:
    • Playful cats are most compatible with other Playful cats.
    • Independent cats are least compatible with Playful cats and more compatible with other Independent cats.
  2. HairLength:
    • Cats with similar hair length (Short-Short, Long-Long) are more compatible.
    • Cats with opposite hair length (Short-Long, Long-Short) are less compatible.

API Endpoints:

  1. GET /cats: This endpoint should return a JSON list of all available cat breeds (all Cat objects).
  2. POST /cats/sort: This endpoint should accept a JSON list of Cat objects in the request body. It should sort the list using the SortCatsByCompatibility algorithm and return the sorted JSON list.

Example Usage:

  • Get all cats:
    Bash
    curl http://localhost:5000/cats
  • Sort cats by compatibility:
    Bash
    curl -X POST http://localhost:5000/cats/sort -H "Content-Type: application/json" -d '[{"Name":"Abyssinian", "Temperament":"Playful", "HairLength":"Short"}, {"Name":"Persian", "Temperament":"Independent", "HairLength":"Long"}]'

Bonus Points:

  • Implement unit tests for your SortCatsByCompatibility algorithm.
  • Add additional sorting criteria (e.g., energy level) to the algorithm.
  • Extend the API to support filtering cats based on specific criteria.

Evaluation:

This assignment will be evaluated based on the following factors:

  • Correctness and efficiency of the SortCatsByCompatibility algorithm.
  • Functionality and clarity of the API implementation.
  • Code quality, adherence to best practices, and proper error handling.
  • Use of unit tests to ensure code functionality.

Time Estimate:

This assignment should take an experienced C# developer approximately 45 minutes to 1 hour to complete.

Remember:

  • Use clear and descriptive variable names.
  • Implement proper error handling for API requests.
  • Consider using libraries like Newtonsoft.Json for JSON serialization/deserialization.
